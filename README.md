# Worker Productivity Dashboard

# Description
This project is a web-based dashboard to monitor worker productivity using event data.

# Tech Stack
- Django
- Django REST Framework
- HTML, JavaScript
- Docker

# Features
- Add events using API
- Calculate worker metrics (working, idle, production)
- Display data in dashboard table

# How to run locally
pip install -r requirements.txt
python manage.py runserver

# Live URL
https://worker-dashboard-k840.onrender.com

# Note
The deployed application uses a fresh database. Sample data can be added via the admin panel.
